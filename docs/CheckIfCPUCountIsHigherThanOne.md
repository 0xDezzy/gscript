# `CheckIfCPUCountIsHigherThanOne()`

Check CPU core count. Many sandboxes use only one CPU core.

## Argument List

 * None.

## Return Type

 * `obj.areWeInASandbox` (bool) - true if count equals one, otherwise false.

## Example

```js
var obj = CheckIfCPUCoreCountIsHigherThanOne()
// obj.areWeInASandbox
```
