# `DelRegKeyvalue,(registryString, path, value)`

Delete a registry key value.

## Argument List

 * `registryString` (string) - Registry string.
 * `path` (string) - Path.
 * `value` (string) - Value.

## Return Type

 * `obj.runtimeError` (error) - Error.

## Example

```js
var obj = DelRegKeyValue(registryString, path, value)
// obj.runtimeError
```
